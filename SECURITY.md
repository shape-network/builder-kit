# Security Policy

## Supported Versions
Only the latest default branch state and latest release are supported with security fixes.

## Reporting a Vulnerability
Please do not open public issues for security reports.

Use one of these private channels:
- GitHub Security Advisory in this repository
- Email: hello@shape.network

Include:
- affected component/version
- reproduction steps
- impact summary
- suggested fix (if available)

We will acknowledge reports promptly and coordinate disclosure after a fix is available.
